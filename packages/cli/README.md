# @vellora/cli

The `vellora` command line: `render` / `lint` / `fix` for the
[vellora](https://github.com/diomalta/vellora) HTML→PDF renderer.

> **Pre-release (alpha), stub** — the commands arrive in a later release.

## Install

```bash
npm install -g @vellora/cli
```

## Usage

```bash
vellora --help
```
